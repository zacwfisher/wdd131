Tech Portfolio Site Plan

Zac Fisher

WDD 131-A2

Overview

Purpose

My personal website aims to showcase my technical skills, projects, and experience to potential employers and collaborators. It will serve as a dynamic portfolio, demonstrating my abilities and providing a platform for professional networking.

Audience

The intended audience includes recruiters, hiring managers, fellow developers, and anyone interested in my technical work.

Dynamic elements

I'll use JavaScript to create interactive project carousels or galleries, allowing users to explore my projects with dynamic transitions and pop-up details.I'll implement animated skill bars that visually represent my proficiency in different technologies, updating dynamically based on data. I'll use JavaScript to validate the contact form, providing real-time feedback to users and ensuring data integrity. I will use javascript to allow users to filter the displayed projects based on technology used, or sort them by date. I will add a toggle to allow users to switch between light and dark themes. I will also use Javascript to fetch project data from JSON files, and display it on the webpage.
